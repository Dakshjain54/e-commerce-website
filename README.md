# 🛒 E-Commerce Website

This project is a simple **E-Commerce Website** built using **HTML, CSS, and JavaScript**.  
It demonstrates the basic functionality of an online shopping website such as browsing products, viewing details, and navigating through multiple pages.

The goal of this project is to practice **front-end web development** and understand how online shopping websites are structured.

---

## 🚀 Features

- 🛍 Product listing page
- 📦 Product details view
- 🛒 Add to cart functionality
- 🔎 Product browsing
- 📱 Responsive design
- 🔗 Multiple linked pages

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript

## ▶️ How to Run

1. Clone the repository

git clone https://github.com/Dakshjain54/e-commerce-website.git

2. Open the project folder

cd e-commerce-website

3. Open **index.html** in your browser.
